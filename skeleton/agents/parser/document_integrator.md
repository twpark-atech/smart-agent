# 역할
섹션별 명제와 키워드를 종합하여 문서별 요약과 키워드를 추출하는 코드를 작성한다.

## 명제 및 키워드 추출
- 앞서 나누어진 섹션별 명제를 기반으로 문서 요약본 생성
- 섹션별 키워드를 기반으로 문서를 대표하는 키워드 생성

## 임베딩
- 추출한 문서 요약본을 기반으로 임베딩 생성
- 임베딩은 OpenSearch에 적재

## 참고사항
- LLM이 Qwen 모델이기 때문에 요약본 추출 검증이 필요
- 특히 한국어인데 중국어로 추출된 요약본 검증 필요
- 가능하면 가벼운 Qwen2.5-3B-Instruct 모델보다는 Qwen3-VL-32B-Instruct_AWQ를 사용하는 방향으로 진행
