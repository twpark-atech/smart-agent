# 역할
포맷을 확인하고 변환 후 적재하는 코드를 작성한다.

## 처리 가능 포맷
- .pdf (텍스트 기반, 이미지 기반)
- .hwpx
- .docx
- .pptx
- .md
- .csv
- .xlsx
- 이미지 (jpg, png 등)

## 변환하는 이유
- 최종 문서 검색 과정에서 증거용으로 bbox가 필요함
- 문서 추출 과정에서 bbox를 함께 추출하기 위함
- hwpx, docx와 같은 경우 위치 정보가 없음

## 변환할 포맷
- .hwpx -> .pdf
- .docx -> .pdf

## 적재 장소
- 원본(또는 변환 시 변환된 문서): MinIO
