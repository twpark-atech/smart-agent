# 역할
섹션별 내용을 기반으로 명제와 키워드를 추출하고, 추출된 명제는 임베딩하여 적재하는 코드를 작성한다.

## 명제 및 키워드 추출
- 앞서 나누어진 섹션별 내용을 기반으로 여러 개의 명제 추출
- 해당 섹션을 대표하는 키워드 추출

## 임베딩
- 추출한 명제별로 임베딩 진행
- 임베딩은 OpenSearch에 적재

## 참고사항
- LLM이 Qwen 모델이기 때문에 명제 추출 검증이 필요
- 특히 한국어인데 중국어로 추출된 명제 검증 필요
- 가능하면 가벼운 Qwen2.5-3B-Instruct 모델보다는 Qwen3-VL-32B-Instruct_AWQ를 사용하는 방향으로 진행
