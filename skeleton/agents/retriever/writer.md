# 역할
Aggregator가 구조화한 내용을 기반으로 최종 답변을 작성하는 Agent에 대한 코드를 작성한다.

## Writer의 역할
- 구조화된 내용 기반 답변 작성
- 출처 정리 및 하단 기록

## 참고사항
- Aggregator의 `structured_content`만 사용하며, 그 외 확인되지 않은 정보는 사용하지 않음
- 답변 작성 후 사용한 출처는 맨 하단에 기록
- 작성 형식은 사용자 질의 유형에 맞게 조정 (요약형 / 상세형 / 표 형식 등)
- 정량적 데이터가 포함된 경우 수치를 명확히 표기

## 출력 형식

```python
{
    "answer": str,          # 작성된 답변 본문
    "sources": list[str]    # 사용한 출처 목록 (답변 하단 표기용)
}
```

## 사용 모델
- Qwen2.5-3B-Instruct (정형화된 구조 기반 작성으로 소형 모델로 충분)
