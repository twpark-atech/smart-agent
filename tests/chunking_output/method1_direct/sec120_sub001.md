# 9. TENTER 기의 운전 순서 > 소분류 2

- 병합 문단 수: 2
- Proposition: 원단은 Box의 over feed roll까지 예인되고, Operator box에서 선 택 스위치를 Single로 설정한 후 Over feed 스위치는 OFF 상태가 되며, 마지막으로 TENTER ON BUTTON을 누르고 SPEED SET BUTTON을 UP하여 체인이 서행하게 된다.

---

- ④ 원단을 입구 Box 의 over feed roll 까지 예인하고 Operator box 의 선 택 스위치를 Single 로 선택하고 Over feed 스위치를 OFF 한다 .
- ⑤ Operator box 의 TENTER ON BUTTON 을  누르고 SPEED SET BUTTON 을 UP 시켜서 체인이 서행하도록 한다 .
