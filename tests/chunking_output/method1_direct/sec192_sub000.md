# 1) A 법 ( 低水壓法 ) 이란 ? > 소분류 1

- 병합 문단 수: 3
- Proposition: 정수압법, 일정수압법 및 누수법은 모두 시험편의 코팅면에서 물이 나올 때의 수위나 시간을 측정하여 누수를 검사하는 방법입니다.

---

- 정수압법 ( 靜 水 壓法 ) : 수준장치의 수위를 상승시켜서 시험편의 코팅면에 3 곳에서 물이 나올 때의 수위를 측정하는 방법
- 일정수압법 ( 一定 水 壓法 ) : 일정수위를 상승시켜 방치한 후 시험편 코팅 면의 3 곳으로 부터 물이 나올 때까지의 시간을 측정하는 방법
- 누수법 ( 漏 水 法 ) : 일정수위를 상승시켜 일정 시간 후에 시험편을 투과한 물의 체적을 (ml) 측정하는 방법 .
