# 11. Discussion : 투습도 (WVP) > 소분류 1

- 병합 문단 수: 2
- Proposition: PU 코팅의 투습도는 ethylene oxide 유닛의 기능을, 그리고 ethylene oxide 농도는 PEG 유기분자 단위의 비율을 나타내는 system 에서의 두 개념이다.

---

- -친수성 PU 코팅의 투습도는 이 system 에서 ethylene oxide units 가 표현하는 기능을 의미한다 .
- -이 system 에서 ethylene oxide 농도는 PEG soft segment 의  비율을  의미한다 .
