# 2) B 법 ( 高水壓法 ) 이란 ? > 소분류 1

- 병합 문단 수: 5
- Proposition: 정수압법과 일정수압법은 시험편 코팅면에서 물이 나올 때의 수압을 측정하고, 누수법은 일정수압을 유지한 후 시험편을 통과한 물의 양을 측정하며, 두 법칙의 단위는 서로 다르고 환산이 가능하다.

---

- 정수압법 ( 靜 水 壓法 ) : 실린더에 물을 넣고 피스톤 핸들에 의해 수압을 가하여 시험편 코팅면의 3 곳에서 물이 나올 때 수압을 측정한다 .
- 일정수압법 ( 一定 水 壓法 ) : 일정수압을 가하여 방치 한 후 시험편 코팅면 의 3 곳으로부터 물이 나올 때까지의 시간을 측정한다 .
- 누수법 ( 漏 水 法 ) : 일정수압을 가하여 일정 시간 후에 시험편을 투과한 물의 체적을 (ml) 측정하는 방법 .
# 단위 : A 법의 정수압법이 mm H2O B 법의 고수압법이 kg/cm 2 또는 kPa
# 단위 환산방법 : B 법의 1 kg/cm 2 = A 법의 10,000 mm H2O
