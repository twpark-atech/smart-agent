# 1) A 법 ( 低 水 壓法 ) 이란 ? > 소분류 1

- 병합 문단 수: 5
- Proposition: 누수법, 정수압법 및 상승수위법은 모두 시험편의 누수를 측정하기 위한 방법입니다.

---

- ▷ ( 靜 水 壓法 ) : 시험장치의  수위를  상승시켜서  시험편의
- 정수압법 coating 면에 3 곳에서 물이 나올 때의 수위를 측정하는 방법
- ▷ ( 一定 水 壓法 ) : 일정수위를 상승시켜 방치한 후 시험편
- 일정수압법 coating 면의 3 곳으로부터 물이 나올때까지의 시간을 측정하는 방법
- ▷ 누수법 ( 漏 水 法 ) : 과한 물의 체적을 측정하는 방법
