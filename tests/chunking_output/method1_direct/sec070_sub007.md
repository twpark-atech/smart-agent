# 나 ) 오바 히팅을 할 경우 (GAS 직화의 경우 ) > 소분류 8

- 병합 문단 수: 2
- Proposition: 기계에서 노즐 마크 발생시에는 펀칭 방식의 경우 슬롯 방식으로 교체하거나, 줄무늬 발생면의 노즐 풍속을 낮추고, 기계 정지 시에는 담파를 사용해 바람을 우회시키는 것이 효과적이다.

---

- ㄴ ) 노즐 마크일 경우 : 펀칭 Type 의 경우 발생하므로 슬롯방식으로 교체 , 혹은 줄무늬 발생면의 노즐풍속을 줄여서 가공 .
바 ) 기계 정지 시 발생되는 노즐 마크 : 노즐마크 방지담파로 바람을 우회 시킨다 .
