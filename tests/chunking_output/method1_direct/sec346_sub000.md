# (2) 습식 , 건식 공정비교 > 소분류 1

- 병합 문단 수: 2
- Proposition: 건식 다공형과 습식 다공형 모두 PU 조성물로 시작하여 각각 다른 과정을 거쳐 최종 제품을 얻는다.

---

- ◆ 건식 다공형 : PU 조성물 (W/O EMULSION) → COATING → 1 차 건 조 ( 有機溶劑 ) → 2 차 건조 ( 水 ) → 후처리 (TENDERING)
- ◆ 습식 다공형 : BASE 층 조성물 → COATING → 건조 → TOP 조성물 → MILLING → COATING → 응고 → 수세 → 건조 → 후처리
