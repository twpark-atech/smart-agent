# 밀도측정 > 소분류 1

- 병합 문단 수: 2
- Proposition: 밀도가 0.8669 g/㎤의 토크레인과 1.564 g/㎤의 카본 탄클로라이드를 사용하여 길이가 85 ㎝인 구배관을 제작하였으며, 관 내의 밀도 범위는 23 ℃에서 1.10~1.18 g/㎤를 유지하였다.

---

- -Toluene(0.8669g/ ㎤ ) 과 carbon tetrachloride(1.564g/ ㎤ ) 로 밀도 구배관 제작 .
- -총 85 ㎝ 인 관의 길이에 밀도 범위가 1.10g/ ㎤ ~1.18g/ ㎤ 의 값을 가지도 록 구배관을 제작 ( 관의 온도는 23 ℃ 유지 )
