# 5) 가공 중간 검사 시 유의사항 > 소분류 2

- 병합 문단 수: 2
- Proposition: 견본 색상과 비교하기 위해 롤별로 샘플을 채취하고 부착하여 상중하 세 곳에서 색상 차이를 확인한 후, 롤 간 차이가 있는 경우 절마다 샘플을 추가로 채취하여 검사를 진행한다.

- ③ Roll 별 색상차이가 나면 절마다 견본을 채취하여 부착하고 검사 기 준에 합격될 경우 진행시킨다 .
